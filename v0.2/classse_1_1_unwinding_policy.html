Detailed Description

Unwinding semantics of a loop.

Such a policy is said to be sound if and only if every found bug does in fact exist. If a policy is not sound, we call it unsound. A policy is said to be complete if and only if it it finds all existing bugs. Otherwise, it is said to be incomplete. Due to Turing's halting problem, there exists no policy that is both sound and complete.
