Detailed Description

template<typename T>
class se::ValueExpr< T >

Type-safe concrete value with an optional symbolic value.

Leaf in the DAG for a concrete and/or an arbitrary value of type T. The combination of a concrete and symbolic value is the basis for single-path (i.e. DART-style) symbolic execution. This is also known as concolic execution.

Remarks:
T should be a primitive type
Immutable
