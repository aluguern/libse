Filter pairs based on key and predicate.

The output is stored in the last argument and is defined to be the set `{ b | (a, b) in R and predicate(b) = true}` where `R` is the set of pairs `T x U`.

Finds pairs for a given key and partitions them according to the predicate.

The output is stored in the last argument and is defined to be the pair of sets `({ b | (a, b) in R and predicate(b) = true}, { b | (a, b) in R and predicate(b) = false}` where `R` is the set of pairs `T x U`.
